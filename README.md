# Apple Music Cli

### Building
1. #### Git clone
2. #### Run CMAKE
### Running
1. Simply run the executable
### Kebinds
* ``` Q``` - Quit
* ``` R``` - Restart song
* ``` K``` - Next Song
* ``` J``` - Prev song
* ``` P``` - Pause song
